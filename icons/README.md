# 应用图标说明

考研打卡助手需要以下尺寸的图标文件，用于PWA和手机安装。

## 所需图标文件

请将以下尺寸的PNG图标文件放入此目录：

1. `icon-72x72.png` - 72×72像素
2. `icon-96x96.png` - 96×96像素  
3. `icon-128x128.png` - 128×128像素
4. `icon-144x144.png` - 144×144像素
5. `icon-152x152.png` - 152×152像素
6. `icon-192x192.png` - 192×192像素
7. `icon-384x384.png` - 384×384像素
8. `icon-512x512.png` - 512×512像素

## 图标生成方法

### 方法一：使用在线工具
1. 访问 [Favicon.io](https://favicon.io/)
2. 上传一个512×512像素的主图标
3. 下载生成的所有尺寸图标
4. 将下载的文件复制到本目录

### 方法二：使用RealFaviconGenerator
1. 访问 [RealFaviconGenerator](https://realfavicongenerator.net/)
2. 上传你的图标
3. 配置各种平台设置
4. 下载生成的文件包
5. 提取并放入本目录

### 方法三：手动创建
如果你有设计工具（如Photoshop、Figma等）：
1. 设计一个512×512像素的方形图标
2. 主题色建议使用：#667eea（紫色渐变）
3. 图标元素：书籍📚、日历📅、打卡✅等
4. 导出为PNG格式，按上述尺寸分别保存

## 临时占位图标

在开发阶段，你可以：
1. 创建一个简单的512×512像素的彩色方块
2. 使用在线工具缩小到各个尺寸
3. 或者直接复制同一个图标文件，重命名为所有尺寸

## 图标设计建议

- **简洁明了**：避免过多细节
- **高对比度**：确保在各种背景下都清晰可见
- **相关主题**：使用学习、教育、打卡相关元素
- **品牌色**：使用 #667eea（主色）或 #764ba2（辅色）

## 更新图标

如果你更新了图标，需要：
1. 替换本目录中的所有图标文件
2. 更新 `manifest.json` 中的图标路径（如果需要）
3. 清除浏览器缓存
4. 重新安装PWA应用
